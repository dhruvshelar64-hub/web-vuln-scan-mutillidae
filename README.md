# Web Application Vulnerability Assessment

This repository contains a detailed vulnerability assessment report for a Metasploitable2 test environment. The assessment demonstrates hands-on web security skills and vulnerability analysis techniques.

## Project Overview

- **Target:** Metasploitable2 Test Environment
- **Assessment Date:** November 28, 2025
- **Assessor:** Dhruv Shelar
- **Scope:** Web Application Security Assessment

## Contents

- [Vulnerability Assessment Report](Vulnerability_Assessment_Report.pdf)
- Screenshots 

## Tools Used

- Nmap
- Metasploit

## Methodology

- Reconnaissance & Discovery
- Scanning & Enumeration
- Vulnerability Analysis
- Reporting

## Key Findings

- Cross-Site Tracing (XST)
- Outdated Apache and PHP versions
- WebDAV information disclosure
- phpMyAdmin exposure
- Directory enumeration
- Missing security headers

## Remediation Recommendations

- Disable HTTP TRACE method
- Upgrade Apache and PHP
- Secure phpMyAdmin access
- Disable unnecessary modules
- Add security headers

## Educational Purpose

This assessment was conducted on an intentionally vulnerable test environment (Metasploitable2) for educational and training purposes.

*All findings and techniques are specific to this lab environment and demonstrate common web security vulnerabilities and attack methodologies.*
